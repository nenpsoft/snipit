# SnipIt

<img src="app/src/main/assets/snipit.png" width="128" height="128">

SnipIt is an Android tool. With the tool, you can annotate a picture with the following shapes [Demo](https://www.youtube.com/watch?v=5lZjX3iBLEU):

- Rectanges
- Ellipses
- Arrows
- Texts
- Erasers
